# How to Test Your Game Online

## Option 1: CodeSandbox (Easiest - Works in Browser)

1. **Go to**: https://codesandbox.io
2. **Click**: "Create Sandbox" → Choose "React"
3. **Upload/Copy these files**:
   - Copy `src/App.js` → paste into `src/App.js` in CodeSandbox
   - Copy `src/index.js` → paste into `src/index.js` in CodeSandbox
   - Copy `package.json` → paste into `package.json` in CodeSandbox
   - Copy `public/index.html` → paste into `public/index.html` in CodeSandbox

4. **For the board image**: 
   - In CodeSandbox, go to `public` folder
   - Click "Upload" and upload `torahpoly_board.png`
   - The game should work!

**Note**: The dice component should work automatically with the dependencies.

If you have GitHub:
